# Changelog

## 0.1.0 — 2026-07-22

- MVP del AI Booking Engine.
- Next.js 14 + Prisma + SQLite + Tailwind.
- Modelo multi-rubro: restaurante, gimnasio, spa, salón, consultorio, coworking.
- Generación de slots, doble-booking bloqueado vía `@@unique`.
- Socket.io para actualizaciones en tiempo real.
- Asistente conversacional con motor LLM OpenAI-compatible.
- Tests de slots.
